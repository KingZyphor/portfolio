# KingZyphor — Developer Portfolio

This is my personal portfolio website built with **Next.js 14**, styled with **Tailwind CSS**, and deployed using **Vercel**.  
It showcases who I am, what I do, and how to contact me — a modern hub for my web development journey.

---

## Tech Stack

-  [Next.js](https://nextjs.org) — React framework for fast, dynamic websites  
-  [Tailwind CSS](https://tailwindcss.com) — Utility-first CSS styling  
-  [Vercel](https://vercel.com) — Hosting and deployment platform  
-  TypeScript (optional) — for cleaner, safer code

---
